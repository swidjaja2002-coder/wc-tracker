Settings

The scanner uses the Anthropic API. Add your key to enable it on this device — leave it blank to log bets manually. The key is stored only on this device and never leaves it except to call Claude.

Set a low monthly spend cap at console.anthropic.com → Settings → Limits so a leaked key can't run up a bill.
